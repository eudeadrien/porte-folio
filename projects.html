Easysave

And finally, my last project for today, a big data project. During this project, the CHU group (Cloud Healthcare Unit) therefore asked us to help it set up its own data warehouse in order to exploit a large amount of data. After performing data processing jobs via Talend on the source files (PostgreSQL, xls, csv) provided to us, we had to create partitioned and bucketed tables to optimize the query. And finally a data structuring in PowerBI was requested. Another successful project with an A.
